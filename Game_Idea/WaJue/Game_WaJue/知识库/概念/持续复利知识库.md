# 持续复利知识库

## 摘要

持续复利知识库的目标是让每次阅读、提问和分析都留下结构化资产。知识不只停留在聊天记录或散乱笔记里，而是进入可链接、可检索、可更新的维基页面，并在之后的工作中被继续使用。

## 复利来自哪里

- **摘要复用**：资料只需深入整理一次，后续从摘要页和概念页读取。
- **链接复用**：相关概念、实体、项目提前建立连接。
- **矛盾记录**：新资料挑战旧结论时，冲突被显式记录。
- **问题沉淀**：好的问答变成页面，不丢在聊天历史里。
- **维护自动化**：大语言模型承担索引、日志、交叉链接等琐碎维护。

## 在游戏项目中的意义

对 `Game_WaJue` 来说，复利知识库可以把灵感、竞品、机制、设定、角色、系统和决策逐步连成一个可演化的设计资料库。后续讨论某个玩法或世界观时，不需要从零回忆，而是基于已有页面迭代。
