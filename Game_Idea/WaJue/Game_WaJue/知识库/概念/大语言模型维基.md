# 大语言模型维基

## 摘要

大语言模型维基是一种由大语言模型持续维护的 Markdown 知识库模式。它不是只在提问时检索原始资料，而是把资料加工成可互联、可更新、可复用的维基页面。随着资料和问题增加，知识库会逐步形成综合观点、交叉链接、矛盾记录和研究路线。

## 核心组成

1. **原始资料层**：保留来源，不轻易改写。
2. **知识层**：由大语言模型写作和维护的综合知识页面。
3. **规则层**：约定目录、页面格式、工作流和质量标准。

## 适合场景

- 长期研究一个主题。
- 维护游戏设计、世界观、系统设计、竞品分析。
- 阅读书籍或论文并逐章沉淀。
- 个人成长、健康、目标、日志等长期追踪。
- 团队项目知识库。

## 成功条件

- 每次新增来源都要整合，而不是只存档。
- 每次重要问题都要尽量沉淀成页面。
- 索引和日志必须保持更新。
- 定期体检，防止命名漂移、孤立页面和过时结论。
