# 程序 说明

## 摘要

- 本页定义 `程序` 职能资料的收纳边界。
- 具体内容优先放入下方子文件夹；当前只建立目录，不批量创建空白内容页。
- 与其他职能强相关的内容，请链接到 [[知识库/项目/职能分工/总览]] 或对应职能说明页。

## 适合放这里

- **工程架构**：项目结构、模块边界、技术选型、依赖、命名和代码规范。
- **玩法实现**：核心玩法逻辑、交互实现、角色/道具/关卡相关代码方案。
- **UI与交互**：界面逻辑、输入、菜单、HUD、弹窗、动效接入和适配。
- **数据与存档**：配置表、存档、状态同步、数据结构、导入导出与版本兼容。
- **工具链与管线**：编辑器工具、构建流程、资源导入、自动化脚本和 MCP/Unity 控制。
- **性能与调试**：性能预算、Profile 记录、Bug 复现、日志、技术债和优化方案。
