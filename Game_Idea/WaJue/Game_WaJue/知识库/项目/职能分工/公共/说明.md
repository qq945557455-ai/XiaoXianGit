# 公共 说明

## 摘要

- 本页定义 `公共` 文件夹的收纳边界：只放跨职能内容，不承载单一职能的专项资料。
- 具体内容优先放入下方子文件夹；当前只建立目录，不批量创建空白内容页。
- 与其他职能强相关的内容，请链接到 [[知识库/项目/职能分工/总览]] 或对应职能说明页。

## 适合放这里

- **里程碑**：阶段目标、版本计划、交付范围、发布日期和验收结果。
- **任务清单**：跨职能任务、负责人、状态、阻塞项和下一步行动。
- **评审记录**：玩法评审、美术评审、技术评审、会议纪要和决策结论。
- **跨职能决策**：影响策划、程序、美术、质量管理或管理排期的共同结论。
