# 美术 说明

## 摘要

- 本页定义 `美术` 职能资料的收纳边界。
- 具体内容优先放入下方子文件夹；当前只建立目录，不批量创建空白内容页。
- 美术参考图、角色图、场景图、切图、导出样例等附件，统一放入 `原始资料/附件/职能分工/美术/` 下的对应子目录。
- 与其他职能强相关的内容，请链接到 [[知识库/项目/职能分工/总览]] 或对应职能说明页。

## 适合放这里

- **美术风格**：整体视觉目标、参考图、色彩、材质、比例、镜头和风格边界。
- **角色**：角色概念、设定图、模型/贴图/骨骼需求、表情和换装。
- **场景**：场景概念、地形、建筑、道具、氛围、光照和关卡美术需求。
- **UI图标**：UI 风格、按钮、图标、字体、界面切图和适配规范。
- **动画特效**：角色动画、环境动画、技能/交互特效、反馈表现和时间轴。
- **素材导出规范**：命名、尺寸、格式、压缩、图集、Prefab 接入和交付检查。
