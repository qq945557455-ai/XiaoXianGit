# 宫崎骏主角与环境图参考

## 摘要

本页给美术同学集中查看“主角参考图 + 对应环境图”。图片均来自 Studio Ghibli 官方作品静止画页面，已下载到本地附件目录，便于在 Obsidian 中离线对照。

使用时不要直接临摹或复刻角色、服装、场景构图，而是提炼：角色剪影、年龄气质、劳动 / 冒险状态、自然与旧机械共存、环境色彩气氛和空间尺度。

## 使用说明

- 附件目录：`原始资料/附件/职能分工/美术/角色/主角设计/宫崎骏主角环境参考/`
- 主要来源：Studio Ghibli 官方作品页与作品静止画页。
- 使用边界：仅作内部美术研究与风格拆解；不要直接用于对外宣传、商用素材、训练素材或成品替换资源。
- 非吉卜力官方静止画来源暂未收录，例如《鲁邦三世：卡里奥斯特罗之城》《未来少年柯南》等，避免来源和版权不清。

## 快速观察要点

1. **主角通常很生活化**：不是一眼“游戏英雄”，而是像真的在世界里生活的人。
2. **服装服务行动**：飞行、劳动、奔跑、修理、探索都有对应的服装便利性。
3. **环境比角色更大**：森林、浴场、天空、城镇、旧机械和海洋都在压出角色的冒险尺度。
4. **伙伴关系很重要**：主角常和动物、非人生命、机械或神秘伙伴绑定。
5. **可爱不是装饰**：可爱来自动作、困境、笨拙、责任感和具体生活细节。

## 对 Game_WaJue 主角的初步美术转译

### 方向 1：挖掘少年 / 少女

- 参考组合：娜乌西卡的生态共感 + 帕祖的动手能力 + 千寻的普通人成长。
- 视觉要点：耐脏工作服、小工具包、头灯 / 护目镜、旧手套、磨损靴子。
- 环境绑定：地下洞穴、矿道、菌类森林、旧机械残骸。

### 方向 2：小型机械师 + 神秘伙伴

- 参考组合：帕祖 / 希达、琪琪 / 黑猫、宗介 / 波妞。
- 视觉要点：主角负责工具和行动，伙伴负责感知、导航或触发地下机关。
- 环境绑定：小工坊、升降机、旧矿车、地下遗迹入口。

### 方向 3：被地下世界选中的普通孩子

- 参考组合：皋月 / 梅、千寻、牧真人。
- 视觉要点：开局非常生活化，后续装备逐步增加。
- 环境绑定：村庄边缘、塌方口、井、废弃矿洞、地下河。

### 方向 4：生态调停者

- 参考组合：娜乌西卡、阿席达卡、小桑。
- 视觉要点：主角身上同时有人类工具和自然痕迹。
- 环境绑定：人类采掘区与地下生态区的边界。
