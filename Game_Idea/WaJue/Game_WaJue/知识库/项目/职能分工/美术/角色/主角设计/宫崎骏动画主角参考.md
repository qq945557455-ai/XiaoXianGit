# 宫崎骏动画主角参考

## 摘要

当前目标不是复制宫崎骏或吉卜力角色，而是提炼可用于 Game_WaJue 原创主角的设计规律：儿童 / 少年视角、自然与机械并存、具体劳动技能、进入异世界后的成长、以及带有温柔但坚定的行动力。

最适合作为第一轮主角参考的是：娜乌西卡、帕祖 / 希达、皋月 / 梅、琪琪、阿席达卡 / 小桑、千寻、苏菲、波妞 / 宗介、牧真人。它们覆盖了“生态共感者、工坊少年、普通儿童、独立职业少女、调停者、异界劳动者、被诅咒的自我重建者、童话生命、创伤探索者”等方向。

## 使用边界

- 可借鉴：人物功能、成长弧线、职业技能、与自然 / 机械 / 异界的关系、服装层次逻辑、剪影简洁度。
- 避免复制：角色名称、具体服装组合、标志性发型、道具造型、镜头构图、官方角色设定图和“宫崎骏 / 吉卜力风格”的直接复刻。
- 后续视觉关键词建议转译为：手绘童话冒险感、温暖自然主义、生活化少年少女、旧机械与自然共生、低魔奇遇。

## 主角设计提炼

### 常见主角类型

1. **自然共感者**：能听懂、理解或安抚非人生命，例如娜乌西卡、波妞。
2. **普通孩子进入异界**：起点不强，靠行动和善意成长，例如千寻、皋月 / 梅、牧真人。
3. **有具体劳动技能的人**：飞行、送货、修理、制造、驾驶、照料，例如琪琪、帕祖、二郎。
4. **夹在两边的调停者**：文明 / 自然、现实 / 异界、人类 / 非人之间，例如阿席达卡、小桑、苏菲。
5. **带伤口但不卖惨的人**：伤口推动行动，但角色仍有温柔和责任，例如苏菲、波鲁克、牧真人。

### 对 Game_WaJue 的可用方向

- **方向 A：挖掘少年 / 少女**  
  一个会修工具、挖土、听懂地下生物声音的孩子。参考帕祖、娜乌西卡、千寻。适合动作探索、资源收集和地下生态主题。

- **方向 B：小型机械师 + 神秘伙伴**  
  主角负责操作旧机械，伙伴负责感知地下遗迹。参考帕祖 / 希达、琪琪 / 黑猫、宗介 / 波妞。适合双能力玩法。

- **方向 C：被地下世界选中的普通孩子**  
  主角原本只是村里孩子，因为一次塌方或发现遗物进入地下世界。参考千寻、梅、牧真人。适合从“不会”到“能承担责任”的成长线。

- **方向 D：生态调停者**  
  地下有危险生物与人类采掘冲突，主角要在采集、保护、修复之间做选择。参考娜乌西卡、阿席达卡、小桑。适合较深主题，但制作复杂度更高。

## 下一步问题

- Game_WaJue 主角年龄段：儿童、少年、青年，还是成人？
- 主角核心动作：挖掘、修理、飞行、采集、送货、照料、战斗、解谜，哪个是第一优先？
- 主角和地下世界的关系：外来者、继承者、守护者、误入者，还是修复者？
- 是否采用双主角 / 主角 + 非人伙伴结构？
