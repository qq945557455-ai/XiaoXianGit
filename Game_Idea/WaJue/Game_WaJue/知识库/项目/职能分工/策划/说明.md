# 策划 说明

## 摘要

- 本页定义 `策划` 职能资料的收纳边界。
- 具体内容优先放入下方子文件夹；当前只建立目录，不批量创建空白内容页。
- 与其他职能强相关的内容，请链接到 [[知识库/项目/职能分工/总览]] 或对应职能说明页。

## 适合放这里

- **核心玩法与循环**：核心玩法、玩家目标、局内循环、局外循环、胜负条件与体验节奏。
- **系统与规则**：游戏系统、规则细节、交互边界、配置表需求与验收口径。
- **关卡与内容**：关卡草图、内容清单、任务设计、难度曲线与投放节奏。
- **数值与经济**：数值模型、资源产出消耗、成长线、价格、平衡性与调参记录。
- **剧情世界观**：世界观、角色设定、阵营、地点、剧情线和文本需求。
- **需求与版本**：版本目标、功能需求、优先级、验收标准和里程碑拆分。
