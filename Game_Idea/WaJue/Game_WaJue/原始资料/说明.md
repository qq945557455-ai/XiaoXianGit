# 原始资料/ — 原始资料层

这里存放知识库的事实来源。原则：**可以新增，尽量不改写**。

## 推荐放置内容

- `原始资料/来源登记/`：网页、论文、视频、访谈、聊天记录、设计文档的来源登记卡。
- `原始资料/附件/`：图片、截图、PDF、音频转写等附件。

## 使用方式

当你要摄入资料时，可以直接说：

> 摄入 `原始资料/来源登记/xxx.md`

或提供一个网址 / 文件，我会：

1. 登记来源。
2. 写 `知识库/来源摘要/` 摘要页。
3. 更新相关概念、实体、项目页。
4. 更新 [[知识库/索引]] 和 [[知识库/日志]]。

## 注意

- 对外部网页默认只保存链接、元数据和摘要，不搬运无许可全文。
- 若来源含图片，建议下载到 `原始资料/附件/`，再用 `![[原始资料/附件/图片名]]` 引用。
